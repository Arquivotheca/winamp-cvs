Winamp Online Services API
Test Web Page

This is a web page to test the different API methods of the Winamp Javascript API. In order to point Winamp to your own online service start Winamp and go to Options->Preferences->Plug-ins->Media Library, choose the plugin for ml_webdev.dll and click "Configure selected plug-in". Set URL to the URL of the main page.

Check Winamp Javascript API for more info on the API methods used on these test web pages.
