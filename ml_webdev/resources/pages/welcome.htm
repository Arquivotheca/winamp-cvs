Welcome to WebDev

Webdev plugin designed to help with process of creating Online Service for Winamp. Plugin uses same browser engine like online media plugin in Media Library and has access to JSAPI2. In short words this is perfect plugin to work with.

Any time you have questions about how to write Online service or just want to share some knowledge with others - Online Documnetation is your best friend.

New to JSAPI? Want to know what is capable of and see how client web interaction with your own eyes, then visit JSAPI2 Test Page.

Your own service always better than others, so if you feel like it is time to change the world - Create Your Own Service and make world a better place.
